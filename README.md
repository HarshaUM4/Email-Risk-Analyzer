# Email Risk Analyzer

## Description
A Python tool that analyzes email content and detects suspicious patterns such as urgent messages, links, and requests for sensitive information.

## Features
- Detects suspicious keywords
- Identifies links in emails
- Flags sensitive information requests
- Provides risk level assessment

## Technologies Used
- Python
- Regular Expressions (re)

## How to Run
1. Save the program as `email_risk_analyzer.py`
2. Open a terminal in the project folder
3. Run:
   python email_risk_analyzer.py
4. Enter the email content when prompted
